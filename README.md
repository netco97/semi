# 🎹音楽を作る人のため&使う人のための音楽ウェブサイト

<img width="983" alt="스크린샷 2024-02-01 오전 10 40 25" src="https://github.com/netco97/semi/assets/101931428/bf9630ff-2585-45a9-bda1-1788ebac10f2">

<br>

## プロジェクト紹介

- 各種メディアを制作するクリエイターは、自     分の作品で使用する効果音やBGMなどを見つけるのに苦労しています。
- 作曲などの音楽的な要素を制作するアーティストは、自分の創作物を消費してくれるクライアントを見つけるのに苦労しています。
- このウェブサイトの目標は、アーティストとアーティストが作った曲を使ってくれるクリエイターの「つながり」を作り上げることです。

## 5. プロジェクト感想

### 🍊 オ・テフン

プロジェクト始まる時にDB設計やページの仕組みなどを一から十まで考えて設計しなければならないことには大変さを感じましたが、
チームの皆さんと協力して一つ一つ解決していったことにとても大きなやりがいを感じています。
何かを作るということは辛いけど楽しいことだということをもう一度感じました。

<br>

### 👻 イ・サンミン

同じことについて話してもお互いに理解することが少しずつ違いますので、後で問題になったことがありました。
経験してみることで、少しはいろいろな状況で対応できるようになりました。
最初の計画段階からもっと詳しくて明示して開発を始めれば、もう少し楽になれると考える良い経験でした。

<br>

### 😎 カン・ウギュ

プロジェクトを始めたばかりの頃は、まだ学んでいないスプリングブーツやタイムリーフなど、不慣れな概念を使わなければならない状況になって、うまくできるか少し恐れがありましたが、プロジェクトを進めていくうちにだんだん慣れてきて、ジャバの実力も以前より伸びたと感じ、新しいことに挑戦することに対する恐怖を少し破ることができる機会になったと思います。

<br>

### 🐬 キム・ドンユル

このプロジェクトでの主な困難は、初めて試みる技術とチーム内のコミュニケーションの難しさでした。新しい技術と慣れない環境での挑戦は大変でしたが、粘り強いな努力を通じてこれを克服することができました。もちろん、同僚たちの支援もとても役に立ちました。 このプロジェクトを終了した今、何よりも満足できる点は、どんな困難にも解決する方法を探そうと努力すれば、解決が可能だという確信を得たということです。良い仲間と楽しい経験をしました。
